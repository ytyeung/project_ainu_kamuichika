沼貝親自吟唱的歌謠「托努佩卡 蘭蘭」

托努佩卡 蘭蘭
強烈的日光將我居住的地方
曬得乾涸，我眼看就要死了。
「誰來給我水喝，
救救我吧。水啊水啊」我們哭喊著，
從遠遠的海濱方向，走來一位女子，
背著竹簍。
我們哭泣著，她從我們身旁經過，
看到我們，說道：
「奇怪的沼貝，可惡的沼貝，哭什麼
吵吵鬧鬧的真煩人。」說著
就踩踏我們，用腳尖踢開，連同貝殼一起碾碎，
徑直往山裡去了。
「好痛，好難受，水啊水啊。」我們哭喊著，
從遠遠的海濱方向，又走來一位女子，
背著竹簍。我們
「誰來給我們水喝，救救我們吧，
好痛，好難受，水啊水啊。」地哭喊著。
於是，那位姑娘，以神祇般美麗高貴的姿態，
來到我們身邊，看到我們，
「哎呀，真可憐，天氣這麼熱，沼貝們的
棲身之所也乾涸了，想要喝水，
這是怎麼了呢？
好像是被踩踏過一樣......」說著
便將我們全都撿起來，放在蕗的葉子上，
放進了清澈的湖水中。
我們喝了清澈的冷水，完全恢復了元氣，
變得非常健康。於是，我們開始
探究她們的性情，
發現先來，踩踏我的
可憎的女子，壞女人是薩瑪雲庫爾的
妹妹，而憐憫我們，
救助我們的年輕姑娘，溫柔嫻靜的
是，奧奇奇里穆伊的妹妹。
薩瑪雲庫爾的妹妹因為心腸惡毒，
使得她的粟米田枯萎，而奧奇奇里穆伊的
妹妹的粟米田則結滿了果實。
那一年，奧奇奇里穆伊的妹妹獲得了大豐收。
得知這是因為我的緣故，
便用沼貝的殼來摘粟米的穗。
從那以後，每年，人類的女子們
在摘粟米穗的時候，都會使用沼貝的殼。
一個沼貝這樣講述著。
```

沼貝親自吟唱了歌謠「托努佩卡 蘭蘭」

托努佩卡 蘭蘭
強烈的日光將我棲身之所
曬得乾涸，我感覺自己快要死了。
「誰能給我們水喝，
請幫助我們。水啊，水啊！」我們哭喊著。
這時，從海灘的方向，走來一位女子，
背著竹簍。
我們哭喊著，她從我們身旁經過，
看到我們，便說：
「奇怪的沼貝，可惡的沼貝，哭什麼哭，
吵吵鬧鬧的！」
她踐踏我們，用腳尖踢打我們，連同貝殼一起踩碎，
然後徑直往山裡去了。
「噢，好痛，好難受，水啊，水啊！」我們哭喊著。
這時，又從海灘的方向，走來一位女子，
背著竹簍。我們哭喊著：
「誰能給我們水喝，救救我們，
噢，好痛，噢，好難受，水啊，水啊！」
於是，那位姑娘，神情美麗而高貴，
來到我身邊，看到我們，便說：
「啊，真可憐，天氣這麼熱，沼貝們的
棲身之所也乾涸了，它們想要喝水。
這是怎麼了？
好像是被踩踏過一樣……」她說著，
將我們全都撿起來，放在蕗葉裡，
然後將我們放進了清澈的湖水中。
我們在清澈冰涼的水中完全恢復了活力，
變得非常強壯。於是，我第一次
探究了那些女子的本性，
結果發現，先來，將我踩碎的
可憎的女子，壞女子，是薩瑪雲庫魯的
妹妹，而憐憫我們，
幫助我們的年輕姑娘，溫柔嫻靜的那位，
是奧基基姆伊的妹妹。
薩瑪雲庫魯的妹妹邪惡可憎，
使她的粟米田枯萎，而奧基基姆伊的
妹妹則使她的粟米田結滿了果實。
那一年，奧基基姆伊的妹妹收穫了大量的粟米。
她知道這是因為我的緣故，
便用沼貝的殼來摘粟米的穗。
從那以後，每年，人類的女子們
在摘粟米穗的時候，都會使用沼貝的殼。
於是，一隻沼貝講述了這個故事。
```
